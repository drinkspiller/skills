# Output Templates

Use the template matching the deliverable type from Phase 1.

--------------------------------------------------------------------------------

## Tagline Only

```markdown
## Tagline Options: [Product/Brand]

**Audience**: [One-sentence portrait from Phase 2]
**Strategic intent**: [What the tagline must accomplish]

### Recommended

> [Tagline]

**Why it works**: [2-3 sentences — Billboard, Swap, and Overnight test results]

### Alternatives

| # | Tagline | Framework Used | Score | Best For |
|---|---------|----------------|-------|----------|
| 1 | [text]  | [framework]    | [X/10]| [context]|
| 2 | [text]  | [framework]    | [X/10]| [context]|

### Killed Directions

- "[rejected tagline]" — [why it failed, which test it flunked]
```

```markdown
### Executions

#### Execution 1: [Title]

#### Execution 2: [Title]

**Copy**: [full copy]
**Visual direction**: [brief note]

#### Execution 3: [Title]
**Format**: [platform/medium]
**Copy**: [full copy]
**Visual direction**: [brief note]

### Unified Thread

[What ties these executions together — shared structure, recurring phrase,
consistent tone, visual system]

### Discarded Executions

- [Rejected direction] — [why it didn't serve the campaign platform]

### Recommendation

[Which execution to lead with and why. Where to test first.]
```
